# Job Board Searcher

> Search popular job boards using Google's Custom Search API.


### Install

```
$ npm install -g jbs
```

```
$ jbs --help

    Search popular job boards using Google's Custom Search API.

    Usage:
        $ jbs <query>

    Example:
        $ jbs "wordpress developer"
```
